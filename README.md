# Análise de Dados das Lojas

Projeto desenvolvido no Google Colab como parte de um challenge de Data Science.

O objetivo é analisar o desempenho de quatro lojas com base em dados de faturamento, categorias de produtos, avaliações dos clientes, produtos mais e menos vendidos e custo médio de frete, a fim de recomendar qual loja deve ser vendida com base nos dados analisados.

Tecnologias utilizadas:
- Python
- Pandas
- Matplotlib
- Google Colab
